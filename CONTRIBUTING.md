# Contributing to SDWorks

Thank you for your interest in contributing to the SDWorks project! 

## How to Contribute

1. **Bug Reports**: If you find a bug, please create an issue on GitHub with a detailed description and steps to reproduce.
2. **Feature Requests**: We welcome new ideas! Open an issue to discuss proposed features.
3. **Pull Requests**:
    - Fork the repository.
    - Create a new branch for your feature or fix.
    - Submit a pull request with a clear description of your changes.

## Code Style

- Keep the UI clean and follow the existing glassmorphism design.
- Use semantic HTML and modern CSS practices.
- Ensure the backend remains compatible with the `diffusers` library.
